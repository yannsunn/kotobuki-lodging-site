## スライド 5: 利用者の困りごと

### コンテンツ
```
宿泊先を探す方々の声

❌ こんなことで困っています

「どこに空室があるかわからない」
• 電話で1軒ずつ問い合わせるしかない
• 満室と言われても次の選択肢がわからない
• 時間と体力を消耗する

「どこに連絡すればいいかわからない」
• 電話番号がわからない
• 営業時間がわからない
• 場所がわからない（地図が読めない）

「サービス内容がわからない」
• Wi-Fiはあるのか？
• 食事は提供されるのか？
• 共同トイレなのか？
```

### ビジュアル提案
- **吹き出し**: 実際の利用者の声を吹き出しで表示
- **イラスト**: 困っている人のイラスト
- **色**: 問題箇所を赤色（❌）で強調

### スピーカーノート
```
利用者の視点から見ると、情報不足は深刻な問題です。

最も多い声が「どこに空室があるかわからない」です。
現在は電話で1軒ずつ問い合わせるしかなく、
満室と言われても次の選択肢がわかりません。

また、連絡先や営業時間、場所がわからないという
基本的な情報さえアクセスしにくい状況です。

さらに、Wi-Fiや食事、設備といったサービス内容も
事前に知ることができず、不安を感じている方が
多くいらっしゃいます。
```

## スライド 6: 事業者の困りごと

### コンテンツ
```
簡易宿泊所経営者の課題

❌ 集客に苦戦しています

「営業力が不足している」
• 広告予算がない
• Webサイトを持っていない（80%以上）
• SNSの活用ができない
• 若年層にリーチできない

「稼働率が低下している」
• 空室があっても知られていない
• リピーターが減少
• 競合との差別化ができない

「情報更新が大変」
• 電話対応に時間を取られる
• 空室情報を都度伝える手間
• 営業時間外の問い合わせに対応できない
```

### ビジュアル提案
- **グラフ**: 稼働率の推移グラフ（下降トレンド）
- **円グラフ**: Webサイト保有率（80%以上が未保有）
- **写真**: 事業者のイメージ写真

### スピーカーノート
```
事業者側も深刻な課題を抱えています。

最も大きいのは営業力の不足です。
80%以上の施設がWebサイトを持っておらず、
広告予算もない中で、どうやって集客すればいいのか
わからない状態です。

その結果、空室があっても稼働率が低いまま。
リピーターも減少し、競合との差別化も
できていません。

また、電話対応に時間を取られ、
空室情報を都度伝える手間も大きな負担です。
営業時間外の問い合わせには対応できず、
機会損失が発生しています。
```

## スライド 7: 課題の構造化

### コンテンツ
```
問題の根本原因

┌─────────────────────────────────┐
│   情報の非対称性                 │
│   (情報がない・届かない)         │
└─────────────┬───────────────────┘
              │
     ┌────────┴────────┐
     ↓                    ↓
┌─────────┐        ┌──────────┐
│ 利用者   │        │ 事業者    │
├─────────┤        ├──────────┤
│• 探せない│        │• 知られない│
│• 選べない│        │• 売れない  │
│• 不安    │   ⇔   │• 低稼働率  │
└─────────┘        └──────────┘
     │                    │
     └────────┬────────┘
              ↓
    ┌──────────────────┐
    │   地域の課題         │
    │ • 高齢化加速         │
    │ • 空き家増加         │
    │ • 地域衰退           │
    └──────────────────┘

💡 解決策: 情報の一元化と可視化
```

### ビジュアル提案
- **図解**: フローチャート形式で因果関係を表示
- **矢印**: 問題の連鎖を示す
- **色分け**: 利用者=青、事業者=緑、地域=赤

### スピーカーノート
```
これらの課題を構造化すると、
根本原因は「情報の非対称性」にあることがわかります。

利用者は情報がないために探せない、選べない、不安。
事業者は情報を発信できないために知られない、売れない、
低稼働率に苦しむ。

この双方の問題が放置されると、
地域全体の高齢化加速、空き家増加、地域衰退という
より深刻な課題につながります。

解決策は明確です。
情報を一元化し、可視化すること。
それが本プロジェクトの出発点です。
```
