# 🏨 宿泊所オーナー様向け - 初回セットアップガイド

このガイドは、パソコンやスマートフォンが苦手な方でも、空室管理システムを使えるように作成しました。

---

## 📱 必要なもの

- スマートフォンまたはパソコン
- インターネット接続
- メールアドレス（携帯メールでもOK）

---

## 🚀 初回セットアップ（管理者が代行可能）

### ステップ1: アカウント作成

#### 📲 スマートフォンの場合

1. **ブラウザを開く**（Safari、Chrome など）
2. 以下のURLにアクセス:
   ```
   https://あなたのサイトURL.com/signup
   ```

3. **情報を入力**:
   - お名前: 例）山田太郎
   - メールアドレス: 例）yamada@example.com
   - 電話番号: 例）045-123-4567
   - パスワード: 6文字以上（忘れないようにメモ！）

4. **「アカウント作成」ボタンをタップ**

5. **完了！**
   - 自動的に管理画面に移動します

---

#### 💻 パソコンの場合

1. **インターネットブラウザを開く**（Edge、Chrome など）
2. アドレスバーに以下を入力:
   ```
   https://あなたのサイトURL.com/signup
   ```

3. **情報を入力**（スマホと同じ）

4. **「アカウント作成」ボタンをクリック**

---

## 🔐 2回目以降のログイン方法

### 📲 スマートフォンの場合

1. ブラウザで以下にアクセス:
   ```
   https://あなたのサイトURL.com/login
   ```

2. **メールアドレスとパスワードを入力**

3. **「ログイン」ボタンをタップ**

4. **完了！**
   - 大きなボタンの管理画面が表示されます

---

### 💡 便利な使い方（スマホ）

#### ホーム画面にショートカットを追加

**iPhoneの場合**:
1. Safariでログインページを開く
2. 下部の「共有」ボタン（□に↑）をタップ
3. 「ホーム画面に追加」をタップ
4. 「追加」をタップ

**Androidの場合**:
1. Chromeでログインページを開く
2. 右上の「︙」（3点メニュー）をタップ
3. 「ホーム画面に追加」をタップ
4. 「追加」をタップ

これで、アプリのようにホーム画面から開けます！

## 🎯 空室数の更新方法（超簡単！）

ログイン後、大きなボタンが表示されます。

```
┌─────────────────────────────────┐
│  ホテル寿荘の空室管理           │
│                                 │
│  現在の空室数                   │
│  ━━━━━━━━━━━━━━              │
│         5                       │
│       室                        │
│                                 │
│     [  - 1室  ]   [  + 1室  ]   │
│                                 │
│     [    満室にする    ]        │
│                                 │
│  最終更新: 2025/11/20 14:30     │
└─────────────────────────────────┘
```

### 操作方法

#### 1部屋空きができた → **「+ 1室」ボタンをタップ**
#### 1部屋埋まった → **「- 1室」ボタンをタップ**
#### 満室になった → **「満室にする」ボタンをタップ**
#### 全室空室になった → **「全室空室」ボタンをタップ**

**それだけです！**

更新すると「✓ 更新しました」と表示されます。

## 📞 よくある質問

### Q1. パスワードを忘れました
**A1.** 管理者（森山さん）にご連絡ください。リセット方法をご案内します。

### Q2. ボタンを押しても反応しません
**A2.** 以下を試してください:
1. インターネットに繋がっているか確認
2. ページを再読み込み（下に引っ張る、またはF5キー）
3. それでもダメなら管理者にご連絡ください

### Q3. 間違えて更新してしまいました
**A3.** すぐに正しい数字に戻してください。もう一度ボタンを押せば直ります。

### Q4. 複数の施設を管理していますが？
**A4.** ログインすると、管理している全施設のボタンが表示されます。それぞれ個別に更新できます。

### Q5. スマホを機種変更しました
**A5.** 新しいスマホで、同じメールアドレスとパスワードでログインすれば使えます。

## 🎓 練習方法

初めての方は、以下の順番で練習してみてください:

1. 「+ 1室」を押してみる → 数字が増える
2. 「- 1室」を押してみる → 数字が減る
3. 「満室にする」を押してみる → 0になる
4. 「全室空室」を押してみる → 最大数になる
5. 実際の空室数に戻す

**失敗しても大丈夫！何度でもやり直せます。**
